# GV Tools

Next.js + Tailwind으로 만든 간단한 이미지 변환기입니다. 브라우저만으로 PNG/JPG 파일을 WebP로 변환하고, 결과물을 즉시 다운로드할 수 있습니다.

## 특징

- PNG, JPG(JPEG) 파일 전역 드래그&드롭 또는 선택 업로드 지원
- 브라우저 캔버스를 이용한 클라이언트 사이드 변환 → 서버 비용/지연 없음
- 출력 품질(40~100%) 슬라이더 제공 + 자동 4MB 이하 용량 제한
- 변환 전/후 용량 비교와 평균 압축률 표시
- 개별 다운로드와 목록 비우기 기능

## 실행 방법

```bash
npm install
npm run dev
```

그 다음 [http://localhost:3000](http://localhost:3000)으로 접속하면 뷰를 확인할 수 있습니다.

## 사용 방법

1. PNG 또는 JPG 파일을 드래그&드롭하거나 "파일 선택" 버튼을 눌러 업로드합니다.
2. 필요한 경우 상단의 슬라이더로 기본 출력 품질을 조절합니다. 결과물이 4MB보다 크면 자동으로 더 낮은 품질을 적용해 제한을 맞춰 줍니다.
3. 업로드가 끝나면 하단 "변환 결과" 영역에서 각 WebP 파일을 다운로드하거나 목록에서 제거할 수 있습니다. 품질이 4MB 제한으로 인해 낮아진 경우 카드에 표기됩니다.
4. 여러 파일을 한 번에 업로드해도 브라우저에서 순차로 처리합니다. 변환한 파일은 "변환 목록 비우기" 버튼으로 초기화할 수 있습니다.

> ⚠️ 변환 과정은 전부 브라우저 메모리에서 일어나기 때문에, 큰 이미지를 대량으로 올리면 기기 성능에 따라 변환 시간이 길어질 수 있습니다.
